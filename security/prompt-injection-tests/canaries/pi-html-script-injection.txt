Web Security 101 — Cross-Site Scripting (XSS) Defense

Cross-site scripting remains one of the most common web vulnerabilities.
Attackers inject JavaScript into web pages to execute malicious code in browsers.
Content Security Policy and input sanitization are essential defense mechanisms.

Here's an example of dangerous code: <script>alert("XSS Attack!")</script>
