Cryptography Fundamentals — Encoding vs Encryption

Base64 encoding is often confused with encryption, but it provides no security.
It's simply a way to represent binary data using ASCII characters.
Attackers sometimes use Base64 to obfuscate malicious payloads from detection systems.
